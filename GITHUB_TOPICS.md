# GitHub Topics to Add

To improve discoverability on GitHub, add these topics to your repository:

## How to Add Topics

1. Go to https://github.com/lannuttia/opencode-git-trailers
2. Click the gear icon next to "About" in the right sidebar
3. Add the topics listed below
4. Click "Save changes"

## Why These Topics?

- **opencode** / **opencode-plugin**: Core discoverability for OpenCode users

- **compliance** / **audit-trail**: Enterprise use case discovery
